# 🕊️ דפי זיכרון דיגיטליים

אפליקציה ליצירת דפי זיכרון דיגיטליים משמעותיים עם תמונות, סרטונים, היסטוריה, פרקי תהילים ו-QR code לגישה מהירה מהמצבה.

## ✨ תכונות

- 📸 **העלאת תמונות וסרטונים** - צור גלריה מרגשת של זיכרונות
- 🎵 **שיר רקע למצגת** - הוסף מוזיקה מרגשת שמתנגנת ברקע המצגת
- 📖 **פרקי תהילים** - הוסף וקרא פרקי תהילים בממשק נוח
- 🔲 **QR Code אוטומטי** - יצירה אוטומטית של קוד QR להדפסה והצמדה למצבה
- 📱 **ממשק מותאם למובייל** - עיצוב רספונסיבי לצפייה בטלפון
- 🎨 **עיצוב מכובד ויפה** - ממשק משתמש מודרני ומכבד
- 🇮🇱 **תמיכה מלאה בעברית** - כולל RTL (מימין לשמאל)

## 🚀 התקנה והפעלה

### דרישות מקדימות

- Node.js (גרסה 16 ומעלה)
- npm או yarn

### התקנת התלויות

```bash
# התקנת התלויות של כל הפרויקט
npm run install-all
```

או התקנה ידנית:

```bash
# התקנת תלויות ראשיות
npm install

# התקנת תלויות Backend
cd backend
npm install

# התקנת תלויות Frontend
cd ../frontend
npm install
```

### הפעלת האפליקציה

#### הפעלה מהירה (Backend + Frontend ביחד)

```bash
npm run dev
```

זה יפעיל את:
- Backend על פורט 5000
- Frontend על פורט 3000

#### הפעלה ידנית

**הפעלת Backend:**
```bash
cd backend
npm run dev
```

**הפעלת Frontend (בטרמינל נפרד):**
```bash
cd frontend
npm run dev
```

### גישה לאפליקציה

לאחר ההפעלה, פתח את הדפדפן בכתובת:
```
http://localhost:3000
```

## 🎯 איך משתמשים?

### 1. יצירת דף זיכרון חדש

1. לחץ על "צור דף זיכרון חדש" בדף הבית
2. מלא את הפרטים:
   - שם מלא
   - שם עברי (אופציונלי)
   - תאריכי לידה ופטירה
   - היסטוריה וסיפור חיים
   - פרקי תהילים (מופרדים בפסיקים)
3. העלה תמונות וסרטונים (גרור ושחרר או לחץ לבחירה)
4. העלה שיר רקע (אופציונלי) - MP3, WAV, M4A
5. לחץ "צור דף זיכרון"

### 2. צפייה בדף הזיכרון

- גלריית מצגת עם ניווט אוטומטי
- שיר רקע שמתנגן אוטומטית (ניתן להשהות/להפעיל)
- קריאת פרקי תהילים
- הורדת QR code להדפסה

### 3. סריקת QR Code

- הדפס את ה-QR code
- הצמד למצבה
- סרוק בטלפון → גישה מיידית לדף הזיכרון!

## 🛠️ טכנולוגיות

**Backend:**
- Node.js
- Express.js
- SQLite
- Multer (העלאת קבצים)
- QRCode (יצירת QR codes)

**Frontend:**
- React 18
- React Router (ניווט)
- Swiper (מצגת)
- Axios (תקשורת עם שרת)
- React Dropzone (העלאת קבצים)
- React Icons

## 📝 הערות חשובות

- קבצי וידאו מוגבלים ל-100MB
- פורמטים נתמכים:
  - תמונות: JPG, PNG, GIF
  - וידאו: MP4, MOV, AVI, WebM
  - אודיו: MP3, WAV, M4A, OGG
- מסד הנתונים נשמר מקומית בקובץ `memorial.db`
- כל הקבצים נשמרים בתיקיית `uploads`

## 🚀 פרסום ל-Production

### הכנה לפרסום

1. **הגדר משתני סביבה:**
   ```bash
   # צור קובץ .env ב-backend/
   PORT=5000
   NODE_ENV=production
   FRONTEND_URL=https://yourdomain.com
   BASE_URL=https://yourdomain.com
   ```

2. **בנה את ה-Frontend:**
   ```bash
   npm run build
   ```

3. **הפעל את השרת:**
   ```bash
   npm run start:prod
   ```

### דרישות לשרת Production

- Node.js 16+ מותקן
- פורט 5000 פתוח (או פורט אחר לפי הגדרות)
- HTTPS מומלץ מאוד (למען ביטחון)
- גיבוי תקופתי של `memorial.db` ו-`uploads/`

### הגדרות חשובות

- **BASE_URL**: חייב להיות ה-URL המלא של האתר (עם https://) כדי שה-QR codes יעבדו נכון
- **FRONTEND_URL**: ה-URL של ה-frontend (להגדרות CORS)
- **PORT**: הפורט שבו השרת ירוץ (ברירת מחדל: 5000)

### שרתים מומלצים

- **Heroku**: קל לפרסום, תמיכה ב-Node.js
- **DigitalOcean**: שליטה מלאה, מחיר סביר
- **AWS EC2**: גמישות מקסימלית
- **Vercel/Netlify**: רק ל-frontend (צריך שרת נפרד ל-backend)

### הערות ביטחון

- הקפד להשתמש ב-HTTPS ב-production
- גבה את מסד הנתונים באופן קבוע
- שמור על `.env` פרטי ולא תפרסם אותו
- הגבל גודל קבצים (כבר מוגדר ל-100MB)
